# Laptop Price Analytics & Value Modeling

## Project Overview

This project analyzes laptop pricing behavior to identify the key hardware factors that drive price and to design a structured value-scoring framework for decision-making.

The work demonstrates end-to-end analytical capability — from structured data transformation and validation to statistical modeling, clustering, and executive-level insights.

---

## Key Objectives

- Identify the primary drivers influencing laptop price.
- Engineer structured hardware features from raw specifications.
- Develop regression models to understand pricing relationships.
- Segment laptops using clustering techniques.
- Design a composite value score to rank configurations.
- Provide business-oriented insights for budget-based decision scenarios.

---

## Data & Processing Approach

The dataset includes laptop configuration and pricing information such as:

- Brand and model
- CPU and GPU specifications
- RAM and storage details
- Display characteristics
- Weight
- Market price

The project follows a controlled data lifecycle:

1. Raw dataset ingestion  
2. SQL-based parsing and validation  
3. Structured feature engineering  
4. Model-ready dataset generation  

All transformation logic is documented and reproducible.

---

## Analytical Techniques Applied

- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression Modeling (Price Prediction)
- Clustering for Market Segmentation
- Composite Specification Scoring
- Scenario Simulation for Budget Optimization
